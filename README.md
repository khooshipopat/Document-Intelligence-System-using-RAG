# Document-Intelligence-System-using-RAG

• Developed an end-to-end Retrieval-Augmented Generation (RAG) based question-answering system to extract
insights from unstructured documents (PDF, DOCX, TXT, web links).  
• Implemented document processing pipeline including text chunking, embedding generation using
sentence-transformers, and efficient similarity search with FAISS vector database.  
• Integrated LLM-based reasoning with retrieved context using LangChain to generate accurate, context-aware
responses for user queries.  
• Built an interactive Streamlit interface enabling users to upload documents and perform real-time semantic search and
question answering.
